Stone Production Simulation

An intelligent, React-based simulation engine for stone factory management and profitability forecasting.

 Features
3D Visualizer: Real-time schematic rendering of stone blocks and production lines.
Risk Engine: Simulates stone cracking, power outages, and labor absenteeism.
P&L Reporting: Generates a full Profit & Loss statement with Chart.js integration.
Printable Reports: Optimized CSS for high-quality industrial PDF reports.

 Tech Stack
 React.js (via Babel)
 Tailwind CSS
 Chart.js
 Lucide-style SVG Icons

 How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No installation or `npm install` required!
